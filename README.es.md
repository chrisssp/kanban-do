## Acerca de Kanban Do

Una aplicación móvil multiplataforma de tablero kanban construida con Expo y React Native Paper. Organiza tareas en columnas, da seguimiento al progreso y gestiona tu flujo de trabajo con una interfaz Material Design limpia.

## Funcionalidades

- Tablero kanban con gestión de tareas
- Crear, editar y eliminar tareas
- Validación de formularios con react-hook-form + yup
- UI Material Design con React Native Paper
- Estilos con NativeWind
- Multiplataforma (iOS, Android)

## Inicio rápido

### Requisitos previos

- Node.js 18+
- Expo Go (en dispositivo) o Android Studio / Xcode
- npm

### Instalación

```bash
git clone https://github.com/chrisssp/kanban-do.git
cd kanban-do
npm install
```

### Desarrollo

```bash
npx expo start
```

Escanea el código QR con Expo Go, o presiona `a` para emulador Android / `i` para simulador iOS.

### Pruebas

```bash
npm test
```
